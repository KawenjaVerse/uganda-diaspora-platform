Uganda Diaspora App – Full System Development Instructions

We are building a modern, scalable, professional Uganda Diaspora digital platform consisting of:

A Flutter Mobile Application (Android + iOS)
A Web Admin Dashboard
Backend APIs and Database
Push Notification System
CMS-style content management

The platform represents the Uganda Diaspora Unit and should feel official, modern, patriotic, informative, and interactive.

The agent should think creatively and add useful diaspora-focused features where necessary.
